## 连接数据库服务器

`mysql -uroot -p`

> 输入方式也可以写成这样的形式：`mysql -u root -p`
>
> - -u表示是我们的账号，这个账号是`Mysql`软件的账号（不是当前登录系统的账号）
>
> - -p表示要输入我们的密码
>
> 回车，然后输入密码，输入密码后就会进入`MySQL`的操作管理界
>
> 该语句还有两个需要了解的参数
>
> - -P表示我们可以指定连接的端口，`Mysql`的默认端口是3306，不添加的时候，默认使用3306这个端口
> - -h表示连接到数据库的`ip`地址，默认情况下是连接到我们的本地主机（127.0.0.1）
